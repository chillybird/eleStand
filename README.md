# 舵机控制 (MG90S 360° 连续旋转)

基于 ESP-IDF 和 LEDC 外设，控制连接在 ESP32 开发板 D15 引脚的 MG90S 360° 连续旋转舵机。通过时间模拟角度控制，每 3 秒旋转 30°，从 0° 到 180° 循环。

## 舵机类型与控制原理

### 舵机分类

| 类型 | 代表型号 | 控制量 | 脉宽含义 |
|------|----------|--------|----------|
| 位置舵机 | SG90 (180°) | **角度** | 0.5ms=0°, 1.5ms=90°, 2.5ms=180° |
| 连续旋转舵机 | MG90S (360°) | **转速和方向** | 1.5ms=停止, <1.5ms=顺时针, >1.5ms=逆时针 |

### PWM 控制原理

舵机使用 50Hz（周期 20ms）的 PWM 信号控制。脉宽长度决定行为：

- **位置舵机**：脉宽直接对应目标角度，舵机内部闭环控制到达该位置
- **连续旋转舵机**：脉宽偏离 1.5ms 越多转速越快，无法控制绝对角度

### 本项目控制方式

由于 360° 舵机无法直接设定角度，采用**基于时间的角度模拟**方法：

1. 输出旋转脉宽，让舵机以固定速度转动
2. 根据 `转速(°/s) × 时间(s) = 角度(°)` 计算所需转动时间
3. 到达时间后输出停止脉宽
4. 软件端累计角度，到达 180° 后归零重来

## 硬件接线

```
ESP32 开发板          MG90S 舵机
┌─────────┐          ┌──────────┐
│   GND   ├──────────┤ 棕色(地)  │
│   5V    ├──────────┤ 红色(电源) │  ← 也可用外部 5V 电源
│ D15(GPIO15)├───────┤ 橙色(信号) │
└─────────┘          └──────────┘
```

### ⚠️ 接线注意事项

1. **必须共地**：如果用外部 5V 电源，电源 GND 必须和 ESP32 的 GND 连接在一起
2. **不要用 3.3V 供电**：SG90/MG90S 工作电压为 4.8V~6V，3.3V 电压不足，舵机会抖动但无法旋转
3. **不要用 GPIO 引脚供电**：GPIO 输出电流仅约 40mA，舵机堵转电流可达 700mA，会烧毁引脚
4. **信号线可直连 GPIO**：ESP32 的 3.3V 逻辑电平足以驱动舵机信号输入
5. **共地示意图**（使用外部电源时）：

```
外部5V正极 ──── 舵机红线
外部5V GND ─┬── 舵机棕线
            └── ESP32 GND
ESP32 D15  ──── 舵机橙线
```

## 构建与烧录

```bash
# 设置目标芯片（仅首次需要）
idf.py set-target esp32

# 编译
idf.py build

# 烧录并打开串口监视（替换 COMx 为实际端口）
idf.py -p COMx flash monitor
```

## 参数调校

在 `main/hello_world_main.c` 中可调整以下参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `SERVO_GPIO` | `GPIO_NUM_15` | 舵机信号引脚 |
| `ROTATE_PULSE_US` | `1100` | 旋转脉宽(μs)，远离1500=转速更快 |
| `ROTATION_DEG_PER_S` | `300` | 当前脉宽下的近似转速(°/s) |
| `ANGLE_STEP` | `30` | 每步旋转角度 |
| `TOTAL_DELAY_MS` | `3000` | 每步总耗时(ms) |
| `STOP_PULSE_US` | `1500` | 停止脉宽，通常无需修改 |

### 校准步骤

1. 先将 `ROTATE_PULSE_US` 设为 1100，烧录运行
2. 观察实际旋转角度：
   - 旋转过度（超过30°）→ 降低 `ROTATION_DEG_PER_S`（增大旋转时间）
   - 旋转不足（不到30°）→ 提高 `ROTATION_DEG_PER_S`（减小旋转时间）
3. 如需微调转速，也可调整 `ROTATE_PULSE_US`（偏离1500越多越快）

## 常见问题

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 舵机完全不动 | 未共地 / 电源不足 | 检查 GND 连接，确保 5V 供电 |
| 舵机抖动但不转 | 3.3V 供电电压不足 | 改用 5V 供电（开发板5V引脚或外部电源） |
| 一直朝一个方向转 | 脉宽偏离 1.5ms 太远 | 降低转速或使用"时间模拟角度"控制方式 |
| 先正转后反转不停 | 用了位置舵机的代码控制 360° 舵机 | 360°舵机用脉宽控制转速，需使用本项目代码 |
| 无法烧录 / 启动异常 | GPIO15 是启动 strapping 引脚 | 上电时 GPIO15 需为低电平，烧录后再接舵机，或换用其他 GPIO |
